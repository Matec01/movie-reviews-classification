# Movie Reviews Classification Project

A movie reviews classification project supported by AI models.

## Description 

This project was made during my 4th year at ESME engineering school to explore and learn more about Natural Language Processing (NLP) models. 
This repo gathers the several notebooks I made as I was testing several approaches, as well as the app I designed as the final product of the project.
